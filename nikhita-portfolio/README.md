# Nikhita Watpal — Portfolio

Personal portfolio website built with HTML, CSS, and JavaScript.

## 🚀 How to Run

### Option 1: Open directly
Double-click `index.html` to open in your browser.

### Option 2: Live Server (Recommended for VS Code)
1. Open the `nikhita-portfolio` folder in VS Code
2. Install the **Live Server** extension (by Ritwick Dey)
3. Right-click `index.html` → **Open with Live Server**

## 📁 Files
- `index.html` — All sections and content
- `style.css` — Styles, animations, responsive layout
- `script.js` — Scroll effects and interactions
- `README.md` — This file

## 🌐 Deploy to GitHub Pages
1. Create a GitHub repo
2. Upload all files
3. Settings → Pages → Deploy from main branch
4. Live at: `https://[username].github.io/[repo-name]`
